# 🐶 The Dog
> „The Dog“ ist ein WPF-basiertes Videospiel, das von dem Spiel „Snake“ inspiriert wurde und dessen Grundidee aufgreift. Daraus entwickelt es ein eigenes Konzept, bei dem der Spieler einen Hund steuert, und wurde unter dem Namen „The Dog“ veröffentlicht.

## Über das Projekt
„The Dog“ ist ein kleines Arcade-Spiel, bei dem der Spieler einen Hund steuert, der immer länger wird, während er Objekte einsammelt. Das Spiel kombiniert klassische Mechaniken mit einem eigenen visuellen und spielerischen Ansatz

## Features
- Steuerung eines wachsenden Hundes
- Klassisches Arcade-Gameplay
- Einfache & übersichtliche Benutzeroberfläche
- Einstellbare Felgröße, Geschwindkeit und Knochen
